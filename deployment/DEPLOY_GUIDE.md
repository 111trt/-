# LogiGlobe Pro 部署指南 (Aliyun 4C8G)

## 1. 准备工作
您已经拥有阿里云 ECS 服务器（4核 8GB），建议操作系统为 **Ubuntu 20.04/22.04 LTS** 或 **CentOS 7/8**。

本部署包采用 Docker 容器化部署，确保环境一致性且易于管理。

## 2. 上传文件
请将 `deployment` 文件夹中的所有内容上传至服务器的任意目录，例如 `/root/logiglobe`。

推荐使用 SCP 命令或 FileZilla 等工具：
```bash
# 示例 SCP 命令 (在本地终端执行)
scp -r deployment/ root@<您的服务器IP>:/root/logiglobe
```

## 3. 环境安装 (如果服务器尚未安装 Docker)

### Ubuntu:
```bash
sudo apt-get update
sudo apt-get install -y docker.io docker-compose
```

### CentOS:
```bash
sudo yum install -y yum-utils
sudo yum-config-manager --add-repo https://download.docker.com/linux/centos/docker-ce.repo
sudo yum install -y docker-ce docker-ce-cli containerd.io docker-compose-plugin
sudo systemctl start docker
```

## 4. 一键部署
进入上传的目录并启动服务：

```bash
cd /root/logiglobe
docker-compose up -d --build
```

- `up -d`: 后台运行
- `--build`: 强制重新构建镜像

## 5. 验证部署
打开浏览器，访问：
`http://<您的服务器IP>`

您应该能看到 LogiGlobe Pro 的 3D 开场动画。

## 6. 后续维护
- **更新代码**：只需替换 `html/index.html` 文件，无需重启容器（因为配置了 Volume 挂载）。
- **查看日志**：`docker-compose logs -f`
- **停止服务**：`docker-compose down`

## 7. 常见问题
- **端口无法访问**：请检查阿里云 ECS 控制台的 **安全组规则**，确保 **入方向** 已放行 **TCP 80** 端口。
- **地图不显示**：请确保服务器能访问外网（高德地图 API 需要公网连接）。
